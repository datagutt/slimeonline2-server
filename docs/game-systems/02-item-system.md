# Item System

**See:** [`../protocol/04-message-catalog.md`](../protocol/04-message-catalog.md) - Items section

## Item Messages

- MSG_USE_ITEM (35)
- MSG_DISCARD_ITEM (39)
- MSG_DISCARDED_ITEM_TAKE (40)
- MSG_GET_ITEM (41)

## Item Effects

Warp Wing (1): Teleports to spawn  
Sodas (2-5): Restore HP  
Seeds (41-60): Plant trees
